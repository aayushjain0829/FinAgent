# FinAgent 🤖💰

An agentic financial advisor built with **LangGraph** and **Windsurf**. 
FinAgent doesn't just calculate returns; it reasons about portfolio health, 
tax-loss harvesting, and rebalancing based on personal financial goals.

### Tech Stack
- **Engine:** Python 3.10+
- **Orchestration:** LangGraph / LangChain
- **LLM:** Claude 3.7 / GPT-4o
- **Data:** Vector-based retrieval for mutual fund NAVs
